Como paso 1 tenemos:

➼ Empizas abriendo Netbeans en tu computadora.
➼ Selecionas File que se encuetra en la esquina superrior izquierda y selecionas New Project.
➼ Te aparecera una ventana llamada New Project.
➼ Selecionas donde dice Java y Java Aplication que se encuentra al lado derecho.
➼ Pulsa Netx.
➼ Te aparecera otra ventana par colocar el nombre de tu proyecto.
➼ Pulsa Finish.








Como paso 2 tenemos:

➼ Debes hacer click en el nombre de tu proyecto, luego das clik derecho sobre el y selecionas en New luego en Java Class.
➼ Te aparecera la secion Name and Location y en Class Name, colocas el nombre de la clase.
➼ Pulsa Finish.
